Privacy Policy - WorkflowManager

Last Updated: April 5, 2026

WorkflowManager is a macOS desktop application that automates workflow processes and integrates with Google Sheets for data management.

1. Information We Collect

  • Google Account: Email address and Google Sheets access (via OAuth 2.0)
  • Local Data: Configuration settings, workflow history, and application logs stored locally
  • No Tracking: We do NOT collect analytics, tracking data, or share information with third parties

2. How We Use Information

  • Google Sheets Integration: Read and write spreadsheet data according to your workflow rules
  • Local Processing: All workflow processing happens locally on your Mac

3. Data Security

  • OAuth 2.0: Secure Google authentication
  • Local Encryption: Data stored using macOS Keychain
  • HTTPS: Secure API communication

4. Your Rights

  • Access: View all stored data
  • Delete: Remove local data at any time
  • Export: Export your configuration and data
  • Revoke: Cancel Google Sheets access anytime

5. Contact

Email: privacy@workflowmanager.app

6. Compliance

This application complies with Google API Services User Data Policy, GDPR, and CCPA requirements.

This privacy policy is designed to be transparent and user-friendly.
